# GitHub User Finder - Поиск пользователей GitHub

## Информация об авторе

**Имя и фамилия:** Нестеров Сергей

**Дата создания:** Апрель 2026

## Описание программы

GitHub User Finder - это GUI-приложение для поиска пользователей GitHub через официальное API. Приложение позволяет искать пользователей по имени, просматривать детальную информацию, добавлять пользователей в избранное и сохранять список избранного в формате JSON.

### Возможности:

- Поиск пользователей GitHub по имени/логину
- Отображение рейтинга (score) и ссылки на профиль
- Просмотр детальной информации о пользователе (био, репозитории, подписчики и т.д.)
- Добавление пользователей в список избранного
- Удаление пользователей из избранного
- Автоматическое сохранение избранного в JSON-файл
- Автоматическая загрузка избранного при запуске
- Обработка ошибок API и соединения

## Инструкция по запуску

### Требования:

- Python 3.6 или выше
- Tkinter (входит в состав Python)
- Библиотека requests (установка: `pip install requests`)

### Запуск:

1. Откройте терминал или командную строку
2. Перейдите в директорию проекта:
   ```bash
   cd github-user-finder
   ```
3. Запустите приложение:
   ```bash
   python github_user_finder.py
   ```

## Примеры использования

### Поиск пользователя (например, "octocat")
1. Введите имя пользователя в поле поиска:
2. Нажмите кнопку "Поиск" или клавишу Enter
3. В списке отобразятся найденные пользователи с рейтингом и ссылкой

### Добавление в избранное:
1. Выберите пользователя из результатов поиска
2. Нажмите кнопку "⭐ В избранное"
3. Пользователь появится в списке избранного и сохранится в favorites.json

### Просмотр деталей:
1. Выберите пользователя из результатов поиска
2. Нажмите кнопку "📋 Детали"
3. Откроется новое окно с подробной информацией:
   - Имя и логин
   - Биография и локация
   - Количество публичных репозиториев
   - Количество подписчиков и подписок
   - Дата создания аккаунта
   - Ссылка на профиль

### Удаление из избранного:
- Выберите пользователя в списке избранного
- Нажмите кнопку **"🗑️ Удалить из избранного"**
- Пользователь будет удалён из списка и из favorites.json

### Тестовые данные (примеры для проверки):

Попробуйте добавить следующие записи для тестирования:

| Логин | Что проверяем |
|-------|---------------|
| octocat | Демонстрационный аккаунт GitHub |
| mojombo | Один из основателей GitHub |
| torvalds | Линус Торвальдс (создатель Linux) |
| defunkt | Бывший CEO GitHub |
| gjtorikian | Инженер GitHub |

После поиска попробуйте:
- Посмотреть детали любого пользователя (кнопка "📋 Детали")
- Добавить 2-3 пользователя в избранное
- Закрыть и снова открыть приложение - избранное должно сохраниться
- Удалить одного пользователя из избранного
- Попробовать добавить уже существующего в избранном
- Найти несуществующего пользователя (например, "asdfghjkl123456789")

## Валидация данных

- Поле поиска не должно быть пустым
- При выборе пользователя для добавления в избранное необходимо сначала найти его через поиск
- Нельзя добавить одного пользователя в избранное дважды
- Для удаления из избранного необходимо выбрать пользователя в списке избранного
- Для просмотра деталей необходимо выбрать пользователя в результатах поиска
- При ошибках API или соединения выводится соответствующее сообщение

## Структура проекта

```
github-user-finder/
├── github_user_finder.py   # Основной файл приложения
├── README.md               # Документация проекта
├── .gitignore              # Файл игнорирования для Git
```

**Примечание:** Файл `favorites.json` **не включён** в репозиторий, так как он создаётся автоматически при добавлении пользователей в избранное и содержит пользовательские данные.
